### 1.2 The Flow State

Elite performers experience "slow time" - action at high speed, mind in slow motion:

```
Normal cognition:
  DLPFC (critic) active → constant self-monitoring → hesitation → "that won't work"

Flow cognition:
  DLPFC quiets → transient hypofrontality → fearless exploration → "let's try it"
  Neurochemicals: norepinephrine + dopamine + anandamide + endorphins + serotonin

Result:
  Time dilates, pattern recognition accelerates, connections appear
  Action and awareness merge
  The critic is offline
```

### 1.3 The Aha Moment Signature

Research by Kounios & Beeman:

```
300ms BEFORE conscious insight:
  1. Alpha burst over right occipital cortex (sensory gating - block distractions)
  2. Gamma burst (40Hz) in right anterior temporal lobe (connection forming)
  3. ACC fires (anterior cingulate - "there's a better strategy available")

The insight is DETECTABLE before it reaches consciousness.
```

### 1.4 Chunking: The Memory Superpower

```
Normal:             7 ± 2 items in working memory
Chess grandmaster:  50,000-100,000 patterns → each "chunk" holds an entire position
Memory athlete:     80+ digits via hierarchical chunking

The secret: compress complex → simple chunk
Each chunk becomes a single "item"
Build chunks of chunks
Exponential capacity
```

### 1.6 The 10th Man Rule

Israeli intelligence doctrine: If 9 agree, the 10th MUST disagree.

```
Normal team:
  "Looks good" + "I agree" + "Same" → groupthink → blind spot

10th Man rule:
  9 agree → 10th MUST find flaws, MUST argue against
  Not optional
  Forces consideration of what everyone is missing
```
