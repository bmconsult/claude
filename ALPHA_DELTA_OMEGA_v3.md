### 1.2 Key Discovery: Theta-Memory is Neuroscience

From UCLA research:

> "Theta oscillations (4-8 Hz) occur more often during navigation and memory formation. The more quickly a person moves, the more theta oscillations take place."

> "Theta activity has been linked to the encoding of new information and retrieval of memories... Theta-phase locking of single neurons has been observed during spatial memory tasks."

**This isn't metaphor. Θ (Theta) = Memory is neurologically validated.**

### 1.3 Key Discovery: Lambda Calculus IS Computation

From Alonzo Church (1930s):

> "Lambda calculus is a formal system for expressing computation based on function abstraction and application using variable binding and substitution."

> "Lambda calculus is Turing complete—a universal model of computation."

**Λ (Lambda) = Computation is mathematically foundational.**

### 1.4 Key Discovery: Gamma-Epsilon-Mu Form Calibration Triad

From reinforcement learning:

- **Γ (Gamma)**: Discount factor - how much future rewards matter (0 = myopic, 1 = far-sighted)
- **Ε (Epsilon)**: Exploration rate - balance exploit vs explore (ε-greedy algorithm)
- **Μ (Mu)**: Population mean - the expected value, baseline prior

**These three form a complete calibration system.**
