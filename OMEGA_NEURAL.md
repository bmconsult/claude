## Part 1: The Complete Neural Map

### 1.1 The Three-Level Organization

The brain has a hierarchical organization that evolved over time:

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                        EVOLUTIONARY BRAIN LAYERS                                 │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  NEOCORTEX (Newest - ~2 million years)                                          │
│    • Higher cognition, language, reasoning, planning                            │
│    • Conscious thought, abstract thinking                                       │
│    • Unique to mammals, largest in humans                                       │
│                                                                                  │
│  LIMBIC SYSTEM (Mammalian brain - ~150 million years)                           │
│    • Emotion, memory, motivation                                                │
│    • Social behavior, attachment                                                │
│    • Shared with other mammals                                                  │
│                                                                                  │
│  BRAINSTEM + CEREBELLUM (Reptilian brain - ~500 million years)                  │
│    • Survival functions: breathing, heart rate                                  │
│    • Arousal, sleep-wake cycles                                                 │
│    • Basic motor coordination                                                   │
│    • Shared with reptiles, fish                                                 │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## Part 2: Beyond the Brain - The Extended System

### 2.1 The HPA Axis (Stress Response)

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                    HYPOTHALAMIC-PITUITARY-ADRENAL AXIS                          │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  STRESSOR DETECTED                                                              │
│        │                                                                        │
│        ▼                                                                        │
│  HYPOTHALAMUS releases CRH (Corticotropin-Releasing Hormone)                    │
│        │                                                                        │
│        ▼                                                                        │
│  PITUITARY GLAND releases ACTH (Adrenocorticotropic Hormone)                    │
│        │                                                                        │
│        ▼                                                                        │
│  ADRENAL GLANDS release CORTISOL                                                │
│        │                                                                        │
│        ▼                                                                        │
│  EFFECTS:                                                                       │
│    • Increased glucose for energy                                               │
│    • Suppressed immune function                                                 │
│    • Enhanced memory consolidation (acute)                                      │
│    • Impaired working memory (chronic)                                          │
│    • Negative feedback to hypothalamus (stops response)                         │
│                                                                                  │
│  COGNITIVE EFFECTS:                                                             │
│    Acute stress: Enhanced focus, memory encoding                                │
│    Chronic stress: Impaired hippocampus, decision-making, prefrontal function   │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 2.2 The Autonomic Nervous System

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                     AUTONOMIC NERVOUS SYSTEM                                     │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  SYMPATHETIC ("Fight or Flight")                                                │
│    Source: Thoracolumbar spinal cord                                            │
│    Neurotransmitter: Norepinephrine                                             │
│    Effects:                                                                     │
│      • Increased heart rate                                                     │
│      • Dilated pupils                                                           │
│      • Inhibited digestion                                                      │
│      • Increased alertness                                                      │
│      • Energy mobilization                                                      │
│    Cognitive: Focused, narrow attention, quick decisions                        │
│                                                                                  │
│  PARASYMPATHETIC ("Rest and Digest")                                            │
│    Source: Cranial nerves (especially VAGUS) + Sacral spinal cord               │
│    Neurotransmitter: Acetylcholine                                              │
│    Effects:                                                                     │
│      • Decreased heart rate                                                     │
│      • Constricted pupils                                                       │
│      • Enhanced digestion                                                       │
│      • Relaxation                                                               │
│      • Energy conservation                                                      │
│    Cognitive: Broad attention, creative thinking, learning consolidation        │
│                                                                                  │
│  VAGUS NERVE (Cranial Nerve X)                                                  │
│    • 75% of parasympathetic nervous system                                      │
│    • Bidirectional: brain → body AND body → brain                               │
│    • Gut-brain axis communication                                               │
│    • Heart rate variability regulation                                          │
│    • Connects to: NTS → hypothalamus, amygdala, hippocampus, PFC                │
│    • High vagal tone = better emotion regulation, cognition                     │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

```

│                                                                                  │
│  COGNITIVE EFFECTS:                                                             │
│    • Heart coherence → improved cognitive performance                           │
│    • Afferent heart signals influence perception, emotion                       │
│    • "Listen to your heart" has neurological basis                              │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## Part 3: Information Flow and Circuits

### 3.1 Major Processing Loops

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                        MAJOR BRAIN CIRCUITS                                      │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  CORTICO-BASAL GANGLIA-THALAMO-CORTICAL LOOP                                    │
│    Cortex → Striatum → Globus Pallidus → Thalamus → Cortex                      │
│    Function: Action selection, habit learning, reward-based learning            │
│    Multiple parallel loops: Motor, Cognitive, Limbic                            │
│                                                                                  │
│  PAPEZ CIRCUIT (Memory)                                                         │
│    Hippocampus → Fornix → Mammillary bodies → Thalamus → Cingulate → Hippocampus│
│    Function: Episodic memory formation                                          │
│                                                                                  │
│  CORTICO-CEREBELLAR LOOP                                                        │
│    Cortex → Pons → Cerebellum → Thalamus → Cortex                               │
│    Function: Timing, prediction, error correction, procedural learning          │
│                                                                                  │
│  LIMBIC-PREFRONTAL CIRCUIT                                                      │
│    Amygdala ↔ vmPFC ↔ OFC                                                       │
│    Function: Emotion regulation, value-based decision-making                    │
│                                                                                  │
│  DEFAULT MODE NETWORK (DMN)                                                     │
│    Medial PFC + Posterior Cingulate + Angular Gyrus + Hippocampus               │
│    Function: Self-referential thought, mind-wandering, creativity               │
│    Active when NOT focused on external tasks                                    │
│                                                                                  │
│  SALIENCE NETWORK                                                               │
│    Anterior Insula + Dorsal ACC                                                 │
│    Function: Detect important stimuli, switch between DMN and Task-Positive     │
│                                                                                  │
│  CENTRAL EXECUTIVE NETWORK                                                      │
│    DLPFC + Posterior Parietal Cortex                                            │
│    Function: Working memory, attention, goal-directed behavior                  │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────────────────────┐

│    • Integrate information from multiple sources                                │

```

## Part 4: The OMEGA NEURAL Architecture

Now we build the architecture FROM the structures:

### 4.1 Design Principle

**The processes emerge from the structures, not vice versa.**

Each agent or agent-group maps to actual neural structures. The flow follows actual neural circuits.

- **Reproductive**: Not relevant to problem-solving
- **Detailed visual/auditory processing**: Abstracted away
- **Specific neurotransmitter dynamics**: Integrated into functional agents
- **Blood-brain barrier, CSF, etc.**: Infrastructure, not processing
