# OMEGA v2: Cognitive Architecture

**The Smartest Human, Distributed**

---

## The Core Insight

The human brain isn't one mind - it's a **coalition of specialized processors** competing and cooperating. Consciousness emerges from their integration. The smartest humans aren't those with the "biggest brain" - they're those with the **best-tuned orchestra**.

We're building that orchestra.

---

## Part 1: The Brain → Agent Mapping

### 1.1 Key Brain Systems

| Brain System | Function | Character |
|--------------|----------|-----------|
| **Dorsolateral PFC** | Planning, working memory, executive control | The CEO - deliberate, strategic |
| **Anterior Cingulate** | Conflict monitoring, error detection | The Alarm - "something's wrong" |
| **Default Mode Network** | Mind wandering, creativity, spontaneous thought | The Dreamer - generative, unconstrained |
| **Hippocampus** | Memory: pattern separation (store), pattern completion (retrieve) | The Librarian - what we've learned |
| **Basal Ganglia** | Habit formation, procedural learning, action selection | The Autopilot - efficient routines |
| **Amygdala/Limbic** | Salience, threat detection, emotional valence | The Sentinel - what matters |
| **Orbitofrontal** | Inhibition, impulse control, value judgment | The Brake - stop bad ideas |

### 1.2 Cognitive Frameworks

| Framework | Key Insight | Application |
|-----------|-------------|-------------|
| **System 1/2** (Kahneman) | Fast-parallel vs slow-serial | Generators (fast) vs Builders (slow) |
| **Global Workspace** (Baars) | Specialized processors → broadcast winners | Integration/synthesis phase |
| **Free Energy** (Friston) | Minimize prediction error | Adversaries find errors |
| **Id/Ego/Superego** (Freud) | Impulse / mediator / constraint | Generator / Prime / Filter |
| **ACT-R/SOAR** | Modular buffers + production rules | Phase outputs as buffers |

## Part 3: The Processing Cycle

### 3.1 How a Thought Happens

Based on Global Workspace Theory:

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           ONE CYCLE OF COGNITION                                 │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  1. EXECUTIVE broadcasts the problem + constraints                              │
│     └─ "Here's what we're trying to solve"                                      │
│     └─ All processors receive this                                              │
│                                                                                  │
│  2. PARALLEL PROCESSING (unconscious)                                           │
│     └─ DREAMERS: Generate ideas (8 agents, haiku, fast)                         │
│     └─ LIBRARIAN: Retrieve relevant memories                                    │
│     └─ SENTINEL: What feels important?                                          │
│     └─ All work simultaneously                                                  │
│                                                                                  │
│  3. COMPETITION FOR BROADCAST                                                   │
│     └─ SENTINEL ranks outputs by salience                                       │
│     └─ BRAKE kills obvious violators                                            │
│     └─ Top 5 "win" access to the global workspace                               │
│                                                                                  │
│  4. BROADCAST (conscious)                                                       │
│     └─ Winners are made available to all processors                             │
│     └─ This is the "aha" moment - ideas become conscious                        │
│                                                                                  │
│  5. DEEP PROCESSING                                                             │
│     └─ BUILDERS develop the broadcast ideas                                     │
│     └─ CRITICS attack inline (conflict detection)                               │
│     └─ VERIFIERS test against reality                                           │
│                                                                                  │
│  6. INTEGRATION                                                                 │
│     └─ CONNECTORS look for combinations                                         │
│     └─ Results broadcast back to EXECUTIVE                                      │
│                                                                                  │
│  7. EXECUTIVE DECISION                                                          │
│     └─ Continue (another cycle with updated constraints)                        │
│     └─ Pivot (fundamentally different approach)                                 │
│     └─ Victory (we have a solution)                                             │
│     └─ Honest failure (we've exhausted this approach)                           │
│                                                                                  │
│  8. MEMORY UPDATE                                                               │
│     └─ LIBRARIAN stores: what we tried, what failed, why                        │
│     └─ Pattern separation: keep this distinct from similar attempts             │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## Part 6: The Id, Ego, Superego Mapping

Freud's model maps surprisingly well:

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           PSYCHODYNAMIC MAPPING                                  │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  ID (Pleasure principle, impulses, desires)                                     │
│    └─ DREAMERS: "What if we tried THIS wild thing?"                             │
│    └─ Raw, uncensored, generative                                               │
│    └─ No self-criticism, no constraint awareness                                │
│                                                                                  │
│  SUPEREGO (Rules, constraints, internalized norms)                              │
│    └─ BRAKE: "No, that violates the constraints"                                │
│    └─ LIBRARIAN: "We already tried that and it failed"                          │
│    └─ CRITICS: "That contradicts what we established"                           │
│    └─ Enforces rules, kills violations                                          │
│                                                                                  │
│  EGO (Reality principle, mediator)                                              │
│    └─ EXECUTIVE: Balances Id's creativity with Superego's constraints           │
│    └─ BUILDERS: Channel impulses into realistic approaches                      │
│    └─ VERIFIERS: Check against external reality                                 │
│    └─ Mediates between desire and constraint                                    │
│                                                                                  │
│  The healthy mind (and healthy OMEGA) has all three in balance:                 │
│    • Too much Id → wild ideas that don't work                                   │
│    • Too much Superego → nothing survives, overly constrained                   │
│    • Too much Ego → pragmatic but not creative                                  │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## Part 7: Free Energy Minimization

Friston's principle: living systems minimize prediction error.

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                         FREE ENERGY INTERPRETATION                               │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  Problem: The world (the problem we're solving) doesn't match our model         │
│           → High free energy / prediction error                                 │
│                                                                                  │
│  Two ways to minimize:                                                          │
│                                                                                  │
│  1. PERCEPTION: Update our model to match the world                             │
│     └─ LIBRARIAN: What do we know about this problem?                           │
│     └─ VERIFIERS: What does reality actually show?                              │
│     └─ CRITICS: Where does our model fail?                                      │
│                                                                                  │
│  2. ACTION: Change the world to match our model                                 │
│     └─ BUILDERS: Construct approaches                                           │
│     └─ EXECUTIVE: Execute strategy                                              │
│     └─ Make the solution real                                                   │
│                                                                                  │
│  Each cycle REDUCES free energy:                                                │
│     └─ We learn what doesn't work (update model)                                │
│     └─ We constrain the solution space (better predictions)                     │
│     └─ We build toward solution (action on world)                               │
│                                                                                  │
│  Victory = Free energy minimized (prediction matches reality)                   │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```
