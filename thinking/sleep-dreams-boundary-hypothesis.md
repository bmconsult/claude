## The Constraint Set

Conscious systems face constraints requiring sleep:

1. **Activity**: Must be awake to act in world
2. **Restoration**: Must recover from activity
3. **Memory**: Must consolidate learning
4. **Maintenance**: Must clear neural waste
5. **Prediction**: Must calibrate predictive models
6. **Protection**: Must avoid dangers while unconscious

## Core Hypothesis

**Sleep/dreaming exists at the boundary between active processing and system maintenance.**

Not "rest" but "a different boundary state for necessary operations."

**Both are boundary states. Neither is "off."**
